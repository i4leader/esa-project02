## 🎲 游戏玩法

### 基础操作
1. **允许摄像头权限** - 让游戏看到你的手势
2. **手势悬停开始** - 将手放在"START GAME"按钮上3秒
3. **挥手切水果** - 用手势切割飞来的水果
4. **避开炸弹** - 千万别碰黑色炸弹！
